# Backend
## Setup virtual environtment
``` bash
  pip install fastapi uvicorn opencv-python numpy pillow
```

## Run script
``` bash
  uvicorn main:app --reload
```

# Frontend
## Setup virtual environtment
``` bash
  pip install streamlit opencv-python-headless numpy
```
## Run script
``` bash
  streamlit run app.py
```
